classmethod from_qubo(Q, offset=0.0, **kwargs)[source]¶

Create a binary quadratic model from a QUBO model.

Parameters

  • Q (dict) – Coefficients of a quadratic unconstrained binary optimization (QUBO) problem. Should be a dict of the form {(u, v): bias, …} where u, v, are binary-valued variables and bias is their associated coefficient.

  • offset (optional, default=0.0) – Constant offset applied to the model.

Returns

Binary quadratic model with vartype set to Vartype.BINARY.

Return type

BinaryQuadraticModel

Examples

This example creates a binary quadratic model from a QUBO model.

>>> import dimod
>>> Q = {(0, 0): -1, (1, 1): -1, (0, 1): 2}
>>> model = dimod.BinaryQuadraticModel.from_qubo(Q, offset = 0.0)
>>> model.linear    # doctest: +SKIP
{0: -1, 1: -1}
>>> model.vartype
<Vartype.BINARY: frozenset({0, 1})>

ising_interactions()[source]¶

Interactions in the Ising representation QUBO formulation to the Ising formulation We assumption Q is a triangular matrix. H = q ^ T Q q

= 1/4 (1+s) ^ T Q(1+s) = s ^ T Q/4 s + 1 ^ T Q/4 s + s ^ T Q/4 1 + 1 ^ T Q/4 1 = s ^ T nondiag(Q)/4 s + 1 ^ T Q/4 s + s ^ T Q/4 1 + 1 ^ T(Q + diag(Q))/4 1 = sum_{i < j} Q_{ij}/4 s_i s_j

  • sum{i <= j}(Q_{ij} + Q_{ji}) s_i

  • sum_{i <= j}(Q_{ij} + Q_{ii})/4

Therefore

J_{ij} = Q_{ij}/4 h_i = sum{i <= j}(Q_{ij} + Q_{ji})/4 constant_term = sum_{i <= j} Q_{ij}/4 + Tr(Q)(energy bias)
