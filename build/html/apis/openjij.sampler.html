Sampler with Simulated Quantum Annealing (SQA).

Inherits from openjij.sampler.sampler.BaseSampler. Hamiltonian $$H(s) = s H_p + (1-s)sum_i sigma_i^x$$ which $H_p$ is problem Hamiltonian that we want solve.
