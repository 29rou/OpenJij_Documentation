  • schedule_info (dict) – Information about an annealing schedule.

  • iteration (int) – Number of iterations.

sample_ising(h=None, J=None, king_graph=None)[source]¶

Sample from the specified Ising model.

Parameters

  • h (dict) – Linear biases of the Ising model.

  • J (dict) – Quadratic biases of the Ising model.

  • **kwargs – Optional keyword arguments for the sampling method.

Returns

obj:: openjij.sampler.response.Response object.

Examples

This example submits a two-variable Ising problem.

>>> import openjij as oj
>>> sampler = oj.CMOSAnnealer(token="YOUR_TOKEN", iteration=10)
>>> response = sampler.sample_ising({0: 1}, {(0, 1): -1, (1, 2): -1, (0, 80): 3})
>>> print(response)
